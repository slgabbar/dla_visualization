Spectra Visualization with D3.js and Flask

Steps to use tool:

  • Use lower plot to 'pan' over the section of spectra you want to analyze.
  • Use the layer 'drop-down' menu to select the layer you want to visualize.
  • Click "create activations" to create the activations for the spectra input
  • Use your mouse to pan over the input and view highest feature activations!
  • NOTE: If the view rectangle is green, this means the model has identified a DLA in the input.
